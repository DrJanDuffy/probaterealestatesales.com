# Asset Management

Guidelines for images, logos, and media on probaterealestatesales.com.

## Folder Structure

```
public/images/
├── logo/       – Logos, favicon, brand marks
├── brand/      – Agent headshots, team, office
├── locations/  – Location-specific property/area photos
├── icons/      – UI icons, schema graphics
└── general/    – Hero images, multipurpose assets
```
