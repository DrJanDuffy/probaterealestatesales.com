# Hyperlocal Content Guidelines

Content and alt text patterns for probate real estate pages.

## Title Tag Pattern

```
{Service} in {Location} | Nevada Probate | Dr. Jan Duffy
```

**Examples:**
- `Probate Real Estate in Henderson | Nevada Probate | Dr. Jan Duffy`
- `Probate Homes for Sale in Summerlin | Las Vegas Probate | Dr. Jan Duffy`

## Meta Description Pattern

Include: service + location + timeline/cost + phone

```
Expert {service} in {location}, Nevada. {Timeline/cost}. Free consultation: (702) 830-9222
```

## H1 Pattern

```
{Location} Probate {Service}
```

**Examples:**
- Henderson Probate Real Estate Services
- Summerlin Probate Property Sales

## Alt Text Patterns

| Image Type | Pattern | Example |
|------------|---------|---------|
| Location hero | `{Location} Nevada probate property` | Henderson Nevada probate property exterior |
| Agent headshot | `Dr. Jan Duffy, Las Vegas probate realtor` | Dr. Jan Duffy, Las Vegas probate realtor |
| Property | `{Location} {type} probate sale` | Summerlin luxury home probate sale |
| Icon | `{Concept} icon` | Probate court gavel icon |

## Internal Link Anchor Text

Use location + service: "probate real estate in Henderson", "Summerlin probate homes"
