# Request Indexing in Google Search Console

When pages show "Discovered - currently not indexed" in GSC, you can manually request indexing to speed up crawling.

## Steps

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Select the property: **probaterealestatesales.com**
3. Use **URL Inspection** (search bar at top)
4. Paste the URL (e.g. `https://www.probaterealestatesales.com/contact/`)
5. Press Enter
6. Click **Request Indexing**
