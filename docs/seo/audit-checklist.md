# Monthly SEO Audit Checklist

Use this checklist to maintain SEO health for probaterealestatesales.com.

## Schema & Structured Data

- [ ] LocalBusiness schema matches GBP (NAP, hours, address)
- [ ] All pages have valid JSON-LD
- [ ] FAQ schema on FAQ page
- [ ] Review schema at [Google Rich Results Test](https://search.google.com/test/rich-results)

## NAP & GBP Consistency

- [ ] NAP (Name, Address, Phone) identical across site and GBP
- [ ] Business hours match GBP
- [ ] "Call," "Directions," "View Google Reviews" buttons present on key pages

## Technical SEO

- [ ] Sitemap submitted and valid (no redirect URLs in sitemap)
- [ ] Robots.txt allows crawling
- [ ] Canonical URLs correct (https://www.probaterealestatesales.com/...)
- [ ] No redirect chains (single-hop redirects to canonical)
- [ ] Page indexing status in GSC (resolve "Page with redirect" if any)

## Content & Metadata

- [ ] Title tags include location + service
- [ ] Meta descriptions under 160 chars
- [ ] H1s match page intent
- [ ] Alt text on images (location/service keywords)

## Core Web Vitals

- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1
- [ ] Mobile-friendly

## Internal Linking

- [ ] Location anchor text used ("Luxury homes in Summerlin")
- [ ] Key pages linked from homepage
- [ ] Resources and blog linked
