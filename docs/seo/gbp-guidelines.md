# Google Business Profile Guidelines

Ensure site content aligns with GBP for local SEO (2025 guidelines).

## Required on Every Page

1. **NAP** – Name, Address, Phone matching GBP exactly
2. **LocalBusiness schema** – JSON-LD reflecting GBP details
3. **Canonical URL** – https://www.probaterealestatesales.com/...

## Per-Location Pages

Each location page (e.g., `/locations/henderson/`) must have:

- Unique NAP if multi-location (or primary NAP)
- GBP-matching description
- Hours, photos, services
- Map embed with pin
- "Call," "Directions," "View Google Reviews" buttons

## Review & Posts

- Google Reviews widgets where appropriate
- Review schema for star ratings in SERPs
- GBP Posts/offers highlighted on blog or landing pages

## FAQ Sections

- Use typical GBP/audience questions
- Mark up with FAQ schema
- Keep answers concise

## Monthly Actions

1. Audit schema consistency
2. Verify NAP accuracy
3. Check GBP link health
4. Update content if GBP changes
