# .gitattributes Reference

Root `.gitattributes` configuration explained.

## Line Endings

- `* text=auto eol=lf` – Normalize line endings to LF on commit/checkout
- Ensures consistency across Windows/macOS/Linux

## Text Files

- `*.ts text`, `*.tsx text`, etc. – Explicitly mark as text for diff/merge
- Prevents Git from treating them as binary

## Binary Files

- `*.jpg binary`, `*.png binary` – Prevent line-ending changes
- Applied to images, fonts, and similar assets

## Merge Strategy

- `package-lock.json merge=ours` – Prefer local lockfile during merges
- Reduces conflicts when merging branches with dependency changes

## Linguist Overrides

- `*.min.js linguist-generated=true` – Exclude from GitHub language stats
- `.next/** linguist-generated=true` – Exclude build output
- Keeps repo language breakdown accurate
