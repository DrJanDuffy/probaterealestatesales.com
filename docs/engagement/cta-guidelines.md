# Call-to-Action Guidelines

Best practices for CTAs on probaterealestatesales.com.

## Primary CTAs

| CTA | Use Case | Example Copy |
|-----|----------|--------------|
| **Call** | Urgent probate help | "Call (702) 830-9222" |
| **Get Valuation** | Home valuation tool | "Get Free Home Valuation" |
| **Contact** | General inquiry | "Schedule Consultation" |
| **Directions** | Office visit | "Get Directions" |
| **Reviews** | Social proof | "View Google Reviews" |

## Placement Rules

1. **Header/Nav** – Phone number always visible
2. **Hero** – Primary CTA (Call or Get Valuation)
3. **After content sections** – Relevant CTA every 2–3 sections on long pages
4. **Footer** – Call, Contact, Directions, Reviews
5. **Sidebar** (if used) – Sticky CTA on long pages

## Copy Principles

- **Action verbs:** Call, Get, Schedule, Request
- **Benefit-focused:** "Free Valuation," "No Obligation"
- **Urgency when appropriate:** "Get Started Today"
- **Clear next step:** No ambiguity about what happens on click

## Technical Requirements

- `tel:` links for phone numbers
- `href` for external links (e.g., Google Maps, Reviews)
- Button contrast meets WCAG AA
- Mobile tap targets ≥ 44×44px
