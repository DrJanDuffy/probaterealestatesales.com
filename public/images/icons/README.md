# Icons

Probate and real estate icons for UI, schemas, and visual elements.

## Probate Icons

| Icon | Alt Text | Use |
|------|----------|-----|
| Gavel | Probate court gavel icon | Legal/probate sections |
| Home | Probate property icon | Listings, valuation |
| Calendar | Nevada probate timeline icon | 6-8 month timeline |
| Dollar | Probate cost icon | Court costs, fees |

## Expected Files

- `gavel.svg` or `gavel.png` – Probate court (replace Minimalist line art gavel.jpg)
- `home.svg` – Property listings
- `timeline.svg` – Nevada probate process
- Schema/rich result icons

## Usage

- Prefer SVG for scalability
- Keep < 50KB
- Alt: descriptive for meaningful icons; empty for decorative
