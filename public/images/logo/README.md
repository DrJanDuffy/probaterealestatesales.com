# Logo Assets

Store logo files for probaterealestatesales.com – probate real estate / Dr. Jan Duffy.

## Expected Files

- `logo.svg` or `logo.png` – Primary logo
- `logo-light.png` – Light version (for dark backgrounds)
- `logo-dark.png` – Dark version (for light backgrounds)
- `favicon.ico` – Browser favicon (or use root `/favicon.ico`)
- `apple-touch-icon.png` – 180×180 for iOS home screen

## Probate Branding

- Berkshire Hathaway HomeServices Nevada Properties
- Probate Real Estate Sales | Dr. Jan Duffy
- License S.0197614.LLC

## Schema Usage

```json
"logo": "https://www.probaterealestatesales.com/images/logo/logo.png"
```

- PNG fallback for email, social
